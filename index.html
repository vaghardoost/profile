برنامه نویسی وب و اندروید

کارهایی که از انجام آن لذت میبرم

java

زبان برنامه نویسی مستقل از پلتفرم و سیستم عامل که با این زبان میتوان کد های با کیفیت با قدرت پردازش بالا ایجاد کرد

node.js

کامپایلر غنی از کتابخانه های مختلف که امکان برنامه نویسی backend را به زبان جاوااسکریپت فراهم آورده است

javascript

زبان برنامه نویسی صفحات وب که در دستگاه های کلاینت اجرا میشوند و وظایف پردازشی را در کلاینت ها فراهم میاورند

react.js

پراستفاده ترین کتابخانه ی جاوااسکریپت که کدنویسی برای صفحات وب را منظم و ساده تر میکند

mongodb

پایگاه داده ی غیر رابطه ای که از فرم پایگاه داده های کلاسیک متفاوت است و داده ها را به صورت json دخیره میکند

postgresql

پایگاه داده ی رابطه ای قدرتمند که توانایی هماهنگ سازی با بسیاری از فریم ورک های backend را دارد

flutter

فریمورک قدرتمند و نوظهور فلاتر که با آن میتوان اپ قابل اجرا در android و ios و همچنین ویندوز و لینوکس و mac و حتی web به صورت pwa ساخت

nest js

فریمورک سمت backend که از زبان typescript برای ایجاد کد قابل اجرا در node js استفاده میکند

redis

پایگاه داده ای که اطلاعات را بر روی ram ذخیره میکند و بالا رفتن سرعت خواندن و نوشتن داده ها را میلیون ها برابر سریع تر میکند

Apache Kafka

نرم افزار متن باز که تبادل پیام بین اجزای backend را به صورت جریانی از داده ها با سرعت بسیار بالا و امن انجام میدهد

kotlin

زبان برنامه نویسی ای که بین برنامه نویسان اندروید محبوبیت بسیاری را دارد و امکانات گسترده ای را برای برنامه نویسی اندروید داراست

express js

کتابخانه پرطرفدار برای نوشتن هرچه بهتر سرور تحت node.js
